Emoji

Emoji

機能説明

簡単なドット絵を簡単に作成する為のブロックを提供します。各ドットの表示・非表示を簡単に編集できます。

  • Set emoji map in チェックしたマスを、指定した色で塗りつぶします。

  • Set line x row x in 絵文字内の指定した1マスを、指定した色で塗りつぶします。

使用方法

Emoji ブロックをエディタにドラッグして、塗りつぶしたいマスを選択します。
